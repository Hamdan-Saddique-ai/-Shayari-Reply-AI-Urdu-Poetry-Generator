# -Shayari-Reply-AI-Urdu-Poetry-Generator
Shayari Reply AI is a smart Urdu poetry generator that converts any user sentence into beautiful, emotional shayari. It detects mood and context to generate romantic, sad, motivational, or deep poetic replies in seconds, making conversations more expressive and meaningful.
